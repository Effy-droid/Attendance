Vercel 배포 후 발급받은 도메인. https://로 시작.
Vercel 환경변수 AUTH_TOKEN과 동일한 값. 학원장님이 정한 비밀 문자열.

📖 자동 발송 모드 작동 방식

1. 학생이 PIN 입력 → 전송
2. 키오스크가 Vercel API에 발송 요청
3. Vercel이 알리고 API로 SMS/알림톡 발송
4. 부모님 폰에 도착 (3~5초)
5. 키오스크 화면에 "발송 완료" 표시

⚠️ 자동 발송 안 될 때

• 인터넷 연결 확인
• API URL과 토큰이 정확한지 ⚙️ → 자동발송에서 확인
• Vercel 환경변수 (ALIGO_API_KEY, ALIGO_USER_ID, ALIGO_SENDER, AUTH_TOKEN) 점검
• 알리고 잔액 확인 ("잔액 조회" 버튼)
• 자동 모드 OFF로 잠시 전환 후 수동 발송
